## 소개

**바른 한글**은 한국어 맞춤법 검사기의 사용자 경험을 개선하고자 시작된 오픈소스 프로젝트입니다.
기존 시스템의 불편함을 해소하고, 보다 직관적이고 현대적인 UI로 누구나 쉽게 사용할 수 있도록 리디자인된 웹 기반 맞춤법 검사기입니다.
Next.js 기반의 반응형 웹사이트로, 다양한 기기에서 편리하게 이용할 수 있습니다.
